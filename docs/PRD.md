# 영양사

# 영양사 대상 식단 자동 생성·관리 플랫폼 – PRD (Nutritionist Mode)

## 1. 개요 (Overview)

본 제품은 **급식·병원·기업·학교 등에서 근무하는 영양사를 1차 타겟으로 하는
전문 식단 자동 생성 및 관리 플랫폼**이다.

영양사가 기존에 사용하던 식단 데이터를 학습하여
자주 사용되고 검증된 조합을 분석하고,
영양 기준·예산·알레르기·식재료 수급 조건을 동시에 고려한
**월 단위 식단표 초안을 자동 생성**한다.

본 플랫폼은 영양사의 전문적 판단을 보조하는 도구로,
**모든 식단은 영양사의 최종 수정·승인을 전제로 한다.**

---

## 2. 제품 목표 (Goals)

- 영양사의 반복적인 식단 편성 업무 시간 절감
- 급식 기준(영양·예산·알레르기) 동시 충족
- 식재료 변경에 대한 즉각적인 식단 대응
- 엑셀 기반 업무 흐름을 대체·보완하는 실무 친화적 도구 제공

---

## 3. 주요 사용자 (Primary Target User)

### 1차 타겟

- 학교·병원·기업·공공기관 영양사
- 월 단위 단체급식 식단을 편성·관리하는 실무자

### 사용자 특성

- 엑셀 기반 식단 관리 경험 보유
- 식재료 수급 변동에 빈번히 대응
- 가격·영양·알레르기 기준을 동시에 관리해야 함

---

## 4. 핵심 문제 정의 (Pain Points)

- 식단 편성에 반복적으로 많은 시간 소요
- 휴무일·공휴일 반영의 번거로움
- 식재료 변경 시 전체 식단 재조정 부담
- 영양·예산·알레르기 조건의 동시 관리 어려움

---

## 5. 기능 요구사항 (Functional Requirements)

### 5.1 기존 식단 분석 기반 자동 생성

- 영양사가 기존 식단표 업로드
- 분석 항목:
    - 밥 / 국 / 반찬 구성
    - 1인분 기준 가격
    - 열량(kcal)
    - 주요 영양소
    - 알레르기 유발 식품
- 분석 방식:
    - 단순 빈도 분석이 아닌,
    영양 기준 충족도, 영양 균형, 예산 효율성, 운영 안정성을 종합적으로 평가하여
    ‘실무적으로 적합한 식단 조합’을 도출
- 결과:
    - 질적으로 검증된 식단 조합을 기반으로
    다음 달 식단 초안 자동 생성

---

### 5.2 급식일 자동 계산 및 휴무일 반영

- 기본 자동 제외:
    - 주말
    - 공휴일
- 사용자 입력:
    - 재량휴무일
    - 급식 미실시일
- 결과:
    - 실제 급식일 기준 식단만 생성

---

### 5.3 영양사 주도 실시간 식단 수정

- 생성된 식단을 UI 상에서 직접 수정
- 수정 항목:
    - 메뉴 구성
    - 식재료 교체
    - 가격 조정
- 자동 처리:
    - 열량·영양소 재계산
    - 예산 초과 여부 즉시 표시

---

### 5.4 식재료 파일 업로드 및 관리

- 지원 파일 형식:
    - PDF
    - HWP
    - Excel
- 기능:
    - 공급업체 식재료 정보 DB화
    - 가격·공급 가능 여부 관리
- UI:
    - 드래그 앤 드롭 방식으로 식단에 적용
    - 재료 변경 시 식단 자동 갱신

---

## 6. 제외 기능 (Out of Scope)

- 개인 맞춤 다이어트 식단
- 벌크업·건강관리 목적별 식단 모드
- 일반 사용자(B2C) 직접 사용 기능

※ 위 기능은 **영양사 모드 안정화 이후 확장 단계에서 도입 예정**

---

## 7. 비기능 요구사항 (Non-Functional Requirements)

- 영양사 책임 구조를 침해하지 않는 설계
- 모든 계산 결과에 대한 근거 확인 가능
- 기존 엑셀 대비 학습 부담 최소화
- 데이터 이력 관리 및 보안 필수

---

## 8. 기술적 고려사항 (Technical Considerations)

- 식단 분석: 규칙 기반 + 통계 분석
- 파일 파싱: 표 구조 인식 중심
- DB 구성:
    - 식단 이력
    - 식재료 DB
    - 영양 기준 데이터
- 목적별 식단 확장을 고려한 구조 설계

---

## 9. 성공 지표 (KPIs)

- 식단 편성 소요 시간 감소율
- 영양사 수정 빈도 감소
- 월간 활성 영양사 수
- 재사용률

---

## 10. 향후 확장 방향 (Future Expansion)

- 목적별 식단 모드(다이어트, 벌크업, 건강관리)
- 개인 사용자(B2C) 서비스 분리
- 발주·재고 관리 시스템 연계
- 급식 평가 및 보고서 자동 생성
